```typescript
type Event<P, M> = {
  label: string
  payload: P
  meta?: M
}
```

## TherapyConfig

Event sent by the the [ck-therapy-select](20_components/40_ck-therapy-select.md) to configure the contant and behavior of the [ck-therapy-modal](20_components/30_ck-therapy-modal.md).

- Label: `therapy-config`

- Payload:

```typescript
{
  planId: string
  planType: string
  prototypeId: string
}
```
