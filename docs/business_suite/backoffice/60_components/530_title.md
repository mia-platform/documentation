The Title renders a header title.

## How to configure

The title text should be specified using `content` property. [Localized objects][localized-text] are supported.

```json
{
  "tag": "bk-title",
  "properties": {
    "content": {
      "en": "Products list",
      "it": "Listat dei prodotti"
    }
  }
}
```
