## Installation and Authentication

### Installing the Plugin

1. Open Microsoft Excel Desktop or Excel 365
2. Navigate to the **Insert** tab in the ribbon
3. Click on **Get Add-ins** (or **Office Add-ins**)
4. Search for the Data Catalog Excel Plugin in your organization's add-in catalog
5. Click **Add** to install the plugin

### Authentication

1. Once installed, locate the Data Catalog plugin in the Excel ribbon
2. Click on the plugin to open the authentication panel
3. Enter your Data Catalog credentials
4. Upon successful login, you'll see your profile information
5. You can logout at any time from the same panel
