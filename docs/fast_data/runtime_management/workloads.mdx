This section helps you configure either new or existing Fast Data services to communicate with the Control Plane.

:::info Compatibility Matrix
To see if your microservices have support for your Control Plane version, go check the [compatibility matrix section](/fast_data/runtime_management/compatibility_matrix.md).
:::

## Configuration

To enable a Fast Data service to communicate with a Control Plane instance, you have to define a JSON configuration that should
match the one given to the Control Plane.

The JSON object has the same properties for all Fast Data microservices and must be compliant to the following JSON Schema.

<JSONSchemaViewer schema={ControlPlaneSchema} />
